# BBC-News-Scraper
A Python application that retrieves the latest BBC news headlines from RSS feeds and saves them to a CSV file.

Features
- Fetch headlines from BBC RSS feeds
- Choose from Technology, Science, or Business categories
- Display the latest 10 headlines
- Export headlines and links to a CSV file
- Basic error handling

Technologies Used
- Python 3
- Requests
- BeautifulSoup (bs4)
- CSV
- Datetime

How to Run
1. Install dependencies:
   pip install requests beautifulsoup4

2. Run the program:
   python bbc_news_scraper.py

3. Select a category:
   tech
   science
   business

4. View headlines and check the generated CSV file.
